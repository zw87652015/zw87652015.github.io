ImageLayoutManager

Publication-ready figure composer ← Home
Open Source · Apache 2.0

Compose Multi-Panel
Figures with Precision

A PyQt6 desktop application for academics and researchers. Build publication-quality figure layouts with consistent spacing, alignment, labels, and one-click export to PDF, TIFF, or JPG.

Download v2.1.4 View on GitHub
Features
🖼️

Multi-Panel Layouts

Assemble multiple images into a single figure with consistent margins and gaps across all panels.

📐

Hierarchical Cell Splitting

Split layout cells horizontally or vertically to create complex nested arrangements.

🏷️

Label Editing

Per-item label editing with color synchronization so panel labels stay consistent.

📏

Customizable Scale Bars

Add and customize scale bars for each panel — new in v2.1.4.

♻️

Reproducible Layouts

Save and reopen layout definitions (*.figlayout) for reproducible figure generation.

📤

Academic-Friendly Export

WYSIWYG export to PDF, TIFF, and JPG — exactly what you see on the canvas.

🗂️

SVG Support

Import SVG vector images alongside raster formats (PNG, JPG, TIFF) with PDF rendering.

🔄

WYSIWYG Canvas

What you see on screen is exactly what gets exported — no surprises when submitting to journals.
