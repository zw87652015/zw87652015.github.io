Features
Everything you need for journal figures

From layout to export, in a single desktop app.

Σ

LaTeX Support

Render complex formulas, symbols, and chemical notations directly on panels using standard LaTeX syntax.

🖼️

Picture-in-Picture (PiP)

Create insets and magnified views with ease. Overlay sub-images with customizable borders to highlight key details.

📐

Hierarchical Splitting

Split cells infinitely in vertical or horizontal stacks to create sophisticated, multi-tiered layouts.

📤

Vector-Sharp PDF Export

Export publication-ready PDFs with high-DPI image embedding while keeping text and labels as resolution-independent vector objects.

🔄

Smart Drag & Swap

Reorganize panels with intuitive smartphone-style drag-and-drop, featuring smooth spring-physics animations.

📏

Integrated Scale Bar Tool

Add and fine-tune professional scale bars—an essential requirement for microscopy and imaging figures.

🗂️

Nested Layouts

Import existing .figlayout files into any cell to compose complex figures from modular components.

🪄

One-Click Auto Layout

Automatically optimize cell ratios and page dimensions with a single click to perfectly fit all your images.

📷

Batch Grid Import

Select a batch of images and instantly generate a perfectly aligned multi-panel grid, ready for further refinement.

CLI
Headless automation for batch workflows

Render, pack, and inspect layouts from the command line — same quality as the GUI, no display required.

🖥️

Render

Export .figpack / .figlayout to PDF, TIFF, JPG, or PNG with the same renderer as the GUI.

📦

Pack / Unpack

Bundle layouts with assets into .figpack, or extract .figpack to folders.

🔍

Inspect

Query page size, DPI, cell counts, and metadata (text or JSON output).

# Pixel-perfect PDF render at project DPI
imagelayout-cli.exe render figure_4.figpack -f pdf -o figure_4.pdf
# Fast preview PNG with custom DPI
imagelayout-cli.exe render figure_4.figlayout -f png --dpi 150
# Print-ready CMYK TIFF with ICC profile
imagelayout-cli.exe render figure_4.figpack -f tiff --cmyk --icc-profile "path/to/profile.icc" -o fig.tiff

Available only in Windows Installer. See cli.md for full documentation.
